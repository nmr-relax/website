The relax web sites

The main web site for relax is http://nmr-relax.comhttp://nmr-relax.com. From these pages general information about the program, links to the latest documentation, links to the most current software releases, and information about the mailing lists are available. There are also Google search capabilities built into the pages for searching both the HTML version of the manual and the archives of the mailing lists.

The relax web site is hosted by the Gna! project (https://gna.org/https://gna.org/) which is described as ``a central point for development, distribution and maintenance of Libre Software (Free Software) projects''. relax is a registered Gna! project and its primary Gna! web site is https://gna.org/projects/relaxhttps://gna.org/projects/relax. This site contains many more technical details than the main web site.
