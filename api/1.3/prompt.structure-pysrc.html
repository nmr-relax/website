 58          origin located at the centre of mass (of the selected residues).  This diffusion tensor PDB 
 59          file can subsequently read into any molecular viewer. 
 60   
 61          There are four different types of residue within the PDB.  The centre of mass of the 
 62          selected residues is represented as a single carbon atom of the residue 'COM'.  The 
 63          ellipsoidal geometric shape consists of numerous H atoms of the residue 'TNS'.  The axes

 67          will be multiple 'SIM' residues, one for each simulation.  These are essentially the same as 
 68          the 'AXS' residue, representing the axes of the simulated tensors, and they will appear as a 
 69          distribution. 
 70   
 71          As the Brownian rotational diffusion tensor is a measure of the rate of rotation about 
 72          different axes - the larger the geometric object, the faster the diffusion of a molecule. 
 73          For example the diffusion tensor of a water molecule is much larger than that of a 
 74          macromolecule. 
 75   
 76          The effective global correlation time experienced by an XH bond vector, not to be confused 
 77          with the Lipari and Szabo parameter tau_e, will be approximately proportional to the 
 78          component of the diffusion tensor parallel to it.  The approximation is not exact due to the 
 79          multiexponential form of the correlation function of Brownian rotational diffusion.  If an 
 80          XH bond vector is parallel to the longest axis of the tensor, it will be unaffected by 
 81          rotations about that axis, which are the fastest rotations of the molecule, and therefore 
 82          its effective global correlation time will be maximal. 
 83   
 84          To set the size of the diffusion tensor within the PDB frame the unit vectors used to 
 85          generate the geometric object are first multiplied by the diffusion tensor (which has the 
 86          units of inverse seconds) then by the scaling factor (which has the units of second 
 87          Angstroms and has the default value of 1.8e-6 s.Angstrom).  Therefore the rotational 
 88          diffusion rate per Angstrom is equal the inverse of the scale value (which defaults to 
 89          5.56e5 s^-1.Angstrom^-1).  Using the default scaling value for spherical diffusion, the 
 90          correspondence between global correlation time, Diso diffusion rate, and the radius of the 
 91          sphere for a number of discrete cases will be:

 94          |           |               |                   | 
 95          | tm (ns)   | Diso (s^-1)   | Radius (Angstrom) | 
 96          |___________|_______________|___________________| 
 97          |           |               |                   | 
 98          | 1         | 1.67e8        | 300               | 
 99          |           |               |                   | 
100          | 3         | 5.56e7        | 100               | 
101          |           |               |                   | 
102          | 10        | 1.67e7        | 30                | 
103          |           |               |                   | 
104          | 30        | 5.56e6        | 10                | 
105          |___________|_______________|___________________| 
106   
107   
108          The scaling value has been fixed to facilitate comparisons within or between publications, 
109          but can be changed to vary the size of the tensor geometric object if necessary.  Reporting 
110          the rotational diffusion rate per Angstrom within figure legends would be useful. 
111   
112          To create the tensor PDB representation, a number of algorithms are utilised.  Firstly the 
113          centre of mass is calculated for the selected residues and is represented in the PDB by a C 
114          atom.  Then the axes of the diffusion are calculated, as unit vectors scaled to the 
115          appropriate length (multiplied by the eigenvalue Dx, Dy, Dz, Dpar, Dper, or Diso as well as
