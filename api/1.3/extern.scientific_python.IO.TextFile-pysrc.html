 28      """ 
 29      Text files with line iteration and transparent compression 
 30   
 31      TextFile instances can be used like normal file objects 
 32      (i.e. by calling read(), readline(), readlines(), and write()), but 
 33      can also be used as sequences of lines in for-loops. 
 34   
 35      TextFile objects also handle compression transparently. i.e. it is 
 36      possible to read lines from a compressed text file as if it were not 
 37      compressed.  Compression is deduced from the file name suffixes '.Z' 
 38      (compress/uncompress), '.gz' (gzip/gunzip), and '.bz2' (bzip2). 
 39   
 40      Finally, TextFile objects accept file names that start with '~' or 
 41      '~user' to indicate a home directory, as well as URLs (for reading only). 
 42      """
