The NS 3-site linear R1 model

This is the numerical model for 3-site linear exchange using 3D magnetisation vectors. The assumption that kAC = kCA = 0 has been made to linearise this model. It is selected by setting the model to `NS R1rho 3-site linear'. The constraints pA > pB and pA > pC is used to decrease the size of the optimisation space, as both sides of the limit are mirror image spaces. To simplify the optimisation space for the model as in the `NS R1 3-site' model, the assumptions R2A0 = R2B0 = R2C0 = R20 and R1A = R1B = R1C = R1 have been made.
