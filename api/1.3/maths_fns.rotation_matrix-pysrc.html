 66      For the rotation of one vector to another, there are an infinit series of rotation matrices

 68      plane between the two vectors.  Hence the axis-angle convention will be used to construct the 
 69      matrix with the rotation axis defined as the cross product of the two vectors.  The rotation 
 70      angle is the arccosine of the dot product of the two unit vectors. 
 71   
 72      Given a unit vector parallel to the rotation axis, w = [x, y, z] and the rotation angle a, 
 73      the rotation matrix R is:: 
 74   
 75                |  1 + (1-cos(a))*(x*x-1)   -z*sin(a)+(1-cos(a))*x*y   y*sin(a)+(1-cos(a))*x*z | 
 76          R  =  |  z*sin(a)+(1-cos(a))*x*y   1 + (1-cos(a))*(y*y-1)   -x*sin(a)+(1-cos(a))*y*z | 
 77                | -y*sin(a)+(1-cos(a))*x*z   x*sin(a)+(1-cos(a))*y*z   1 + (1-cos(a))*(z*z-1)  | 
 78   
 79   
 80      @param R:           The 3x3 rotation matrix to update. 
 81      @type R:            3x3 numpy array 
 82      @param vector_orig: The unrotated vector defined in the reference frame. 
 83      @type vector_orig:  numpy array, len 3 
 84      @param vector_fin:  The rotated vector defined in the reference frame. 
 85      @type vector_fin:   numpy array, len 3 
 86      """ 
 87   
 88      # Convert the vectors to unit vectors. 
 89      vector_orig = vector_orig / norm(vector_orig)

 91   
 92      # The rotation axis (normalised). 
 93      axis = cross(vector_orig, vector_fin) 
 94      axis_len = norm(axis) 
 95      if axis_len != 0.0:

 97

101      z = axis[2]
