Log:
Visualisation files for all of the pseudo-ellipse permutations by 
frame_order.permute_axes.

This includes the x <= z <= y and z <= x <= y conditions (the previous files 
were for x <= y <= z).
In all permutation combinations, optimisation has been performed to 
demonstrate that these are all
local minima.  These all approximate the rotor when using the CaM frame order 
rotor model synthetic
data.
