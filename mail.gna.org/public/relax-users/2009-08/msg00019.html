Posted by Edward d'Auvergne on August 17, 2009 - 10:54:
Hi,

The 1.3 versions of relax now only uses numpy (this is hardcoded as
numpy imports in the Python code, and library includes for the C
code).  Numeric was listed as a dependency, but this was only because
Scientific python (http://sourcesup.cru.fr/projects/scientific-py/)
required it.  But this orphaned project now has a new lead developer
and the 2.8 and higher versions now use numpy.  So Numeric is no
longer required for relax and has been permanently removed.

Cheers,

Edward
