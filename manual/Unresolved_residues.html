Unresolved residues

As the peaks of certain residues overlap to such an extent that the heights cannot be resolved, a simple text file was created called unresolved in which each line consists of a single residue number. By using the command


deselect.read(name, file=`unresolved')

all residues in the file unresolved are excluded from the analysis.
