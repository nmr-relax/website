Log:
Fixes for the definitions of the 'theta' and 'w_eff' relaxation dispersion 
parameters.

These should not be manually changed by the user and they are not optimised 
parameter.  Therefore
they have been shifted from the set 'params' to the set 'all', to avoid 
listing them in the
parameter tables (for example in the value.set user function).
