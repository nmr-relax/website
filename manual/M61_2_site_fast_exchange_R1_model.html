The M61 2-site fast exchange R1 model

This is the model for 2-site fast exchange for on-resonance R1-type data. It is selected by setting the model to `M61', here named after Meiboom (1961). The equation for the exchange process is

(theparentequation.59)

The reference for this equation is:

  • Meiboom61

More information about the M61 model is available from:

  • the relax wiki at http://wiki.nmr-relax.com/M61,
  • the API documentation at http://www.nmr-relax.com/api/3.1/lib.dispersion.m61-module.html,
  • the relaxation dispersion page of the relax website at http://www.nmr-relax.com/analyses/relaxation_dispersion.html#M61.
