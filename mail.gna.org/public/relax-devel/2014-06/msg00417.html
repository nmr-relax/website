Posted by Edward d'Auvergne on June 23, 2014 - 16:50:
Hi Troels,

In general you need to be careful when deleting an author from the
copyright notice.  Even if there are no lines in common with the
original file, the copyright should not be removed out of respect for
the author.  In this case it is ok, as you are spinning just your code
out of another module.  But if you copy a file as a template and then
modify, the copyright should be retained.  You can see this in the gui
package that was originally written by Michael Bieri.  There is not a
single line of code he wrote left, as I have written the GUI twice,
but his name remains in the copyright notices of all the files he
created or that I copied.

Regards,

Edward
