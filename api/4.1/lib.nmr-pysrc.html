114      # The average resonance offset in the rotating frame. 
115      Delta_omega = chemical_shift - spin_lock_offset 
116   
117      # Calculate the theta angle describing the tilted rotating frame relative to the laboratory. 
118      # theta = atan(omega1 / Delta_omega). 
119      # If Delta_omega is negative, there follow the symmetry of atan, that atan(-x) = - atan(x). 
120      # Then it should be: theta = pi + atan(-x) = pi - atan(x) = pi - abs(atan( +/- x)).
