n_state_model.set_type

Synopsis

Set whether the alignment tensor is the full or reduced tensor.

Defaults

n_state_model.set_type(self, tensor=None, red=False)

Keyword Arguments

tensor: The alignment tensor identification string.

red: The state of the alignment tensor. If True, then it is labelled as the full tensor. If False, then it is labelled as the tensor reduced because of domain motions.

Description

Prior to optimisation of the N-state model the state of alignment tensor, whether it is the full or reduced tensor, must be set using this user function.

Examples

To state that the alignment tensor loaded as `chi3 C-dom' is the reduced tensor, type:


relax> n_state_model.set_type(tensor=`chi3 C-dom', red=True)
