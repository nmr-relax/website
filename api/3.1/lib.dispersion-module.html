The relax-lib NMR package - a library of functions for relaxation dispersion.

Submodules [hide private]
  • lib.dispersion.cr72: The Carver and Richards (1972) 2-site all time scale exchange model (pA > pB).
  • lib.dispersion.dpl94: The Davis, Perlman and London (1994) 2-site fast exchange R1rho model.
  • lib.dispersion.it99: The Ishima and Torchia (1999) 2-site model for all time scales with pA >> pB.
  • lib.dispersion.lm63: The Luz and Meiboom (1963) 2-site fast exchange model.
  • lib.dispersion.lm63_3site: The Luz and Meiboom (1963) 3-site fast exchange model.
  • lib.dispersion.m61: The Meiboom (1961) 2-site fast exchange R1rho model.
  • lib.dispersion.m61b: The Meiboom (1961) 2-site on-resonance skewed population R1rho model.
  • lib.dispersion.mmq_cr72: This function performs a numerical fit of CR72 model extended for MMQ CPMG data.
  • lib.dispersion.mp05: This Miloushev and Palmer (2005) 2-site exchange R1rho model.
  • lib.dispersion.ns_cpmg_2site_3d: This function performs a numerical fit of 2-site Bloch-McConnell equations for CPMG-type experiments.
  • lib.dispersion.ns_cpmg_2site_expanded: This function performs a numerical fit of 2-site Bloch-McConnell equations for CPMG-type experiments.
  • lib.dispersion.ns_cpmg_2site_star: This function performs a numerical fit of 2-site Bloch-McConnell equations for CPMG-type experiments.
  • lib.dispersion.ns_matrices: A collection of functions for generating the relaxation matrices for the numerical solutions.
  • lib.dispersion.ns_mmq_2site: This is the numeric solution for the 2-site Bloch-McConnell equations for mixed CPMG data types.
  • lib.dispersion.ns_mmq_3site: This is the numeric solution for the 3-site Bloch-McConnell equations for mixed CPMG data types.
  • lib.dispersion.ns_r1rho_2site: This function performs a numerical fit of 2-site Bloch-McConnell equations for R1rho-type experiments.
  • lib.dispersion.ns_r1rho_3site: This function performs a numerical fit of 3-site Bloch-McConnell equations for R1rho-type experiments.
  • lib.dispersion.tap03: This is the Trott, Abergel and Palmer (2003) off-resonance 2-site exchange R1rho model.
  • lib.dispersion.tp02: This Trott and Palmer (2002) 2-site exchange R1rho model.
  • lib.dispersion.tsmfk01: The Tollinger et al.
  • lib.dispersion.two_point: The relaxation dispersion equations.
