Details:

To replicate this, use the following steps:

- Open the GUI.
- Select the dx.map user function from the menus.
- Click on the button for changing 'The highlight points in the space'.

What you should see is a list (multiple rows) starting with a single element
(row).  Each row should consist of an index at the start in the first column,
then 3 columns for the {x, y, z} 3D coordinates.

What is seen however is either one column is missing
(http://thread.gmane.org/gmane.science.nmr.relax.scm/21254/focus=5981), though
this is recently fixed.  Now the first column is empty.
