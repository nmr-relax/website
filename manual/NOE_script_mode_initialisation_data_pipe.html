NOE script mode - initialisation of the data pipe

The start of this sample script is very similar to that of the relaxation curve-fitting calculation on page . The command


pipe.create(`NOE', `noe')

initialises the data pipe labelled ``NOE''. The data pipe type is set to the NOE calculation by the argument ``noe''.
