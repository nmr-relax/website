24  """The Luz and Meiboom (1963) 3-site fast exchange model. 
25   
26  This module is for the function, gradient and Hessian of the 'LM63 3-site' model.  The model is named after the reference: 
27   
28      - Luz, S. and Meiboom S., (1963)  Nuclear Magnetic Resonance study of protolysis of trimethylammonium ion in aqueous solution - order of reaction with respect to solvent, J. Chem. Phys. 39, 366-370 (U{DOI: 10.1063/1.1734254<http://dx.doi.org/10.1063/1.1734254>}). 
29   
30  The phi_ex_i and kex_i factors for the 3-site model were derived in the reference: 
31   
32   
33  The equation used is:: 
34   
35                     _3_ 
36                     \    phi_ex_i   /     4 * nu_cpmg         /     ki      \ \  
37      R2eff = R20 +   >   -------- * | 1 - -----------  * tanh | ----------- | | . 
38                     /__     ki      \         ki              \ 4 * nu_cpmg / / 
39                     i=2 
40   
41  For deconvoluting the parameters, see the relax user manual or the reference: 
42   
43      - O'Connell, N. E., Grey, M. J., Tang, Y., Kosuri, P., Miloushev, V. Z., Raleigh, D. P., and Palmer, 3rd, A. G. (2009). Partially folded equilibrium intermediate of the villin headpiece HP67 defined by 13C relaxation dispersion. J. Biomol. NMR, 45(1-2), 85-98. (U{DOI: 10.1007/s10858-009-9340-0<http://dx.doi.org/10.1007/s10858-009-9340-0>}). 
44   
45  """

52      """Calculate the R2eff values for the LM63 3-site model. 
53   
54      See the module docstring for details. 
55   
56   
57      @keyword r20:           The R20 parameter value (R2 with no exchange). 
58      @type r20:              float 
59      @keyword rex_B:         The phi_ex_B / kB parameter value. 
60      @type rex_B:            float 
61      @keyword rex_C:         The phi_ex_C / kC parameter value. 
62      @type rex_C:            float 
63      @keyword quart_kB:      Approximate chemical exchange rate constant between sites A and B (the exchange rate in rad/s) divided by 4. 
64      @type quart_kB:         float 
65      @keyword quart_kC:      Approximate chemical exchange rate constant between sites A and C (the exchange rate in rad/s) divided by 4. 
66      @type quart_kC:         float 
67      @keyword cpmg_frqs:     The CPMG nu1 frequencies. 
68      @type cpmg_frqs:        numpy rank-1 float array 
69      @keyword back_calc:     The array for holding the back calculated R2eff values.  Each element corresponds to one of the CPMG nu1 frequencies. 
70      @type back_calc:        numpy rank-1 float array 
71      @keyword num_points:    The number of points on the dispersion curve, equal to the length of the cpmg_frqs and back_calc arguments. 
72      @type num_points:       int 
73      """ 
74   
75      # Loop over the time points, back calculating the R2eff values. 
76      for i in range(num_points): 
77          # Catch zeros. 
78          if rex_B == 0.0 and rex_C == 0.0: 
79              back_calc[i] = r20 
80   
81          # Avoid divide by zero. 
82          elif quart_kB == 0.0 or quart_kC == 0.0: 
83              back_calc[i] = 1e100 
84   
85          # The full formula. 
86          else: 
87              back_calc[i] = r20 
88              back_calc[i] += rex_B * (1.0 - cpmg_frqs[i] * tanh(quart_kB * cpmg_frqs[i]) / quart_kB) 
89              back_calc[i] += rex_C * (1.0 - cpmg_frqs[i] * tanh(quart_kC * cpmg_frqs[i]) / quart_kC)
