Modified how the relaxation dispersion auto-analysis handles Monte Carlo 
simulations.

Now there is a flag which allows per-model simulations to be enabled.  By 
default, simulations are
now only performed at the end.  This is to allow for massive speed ups in the 
auto-analysis.
