Details:

The user function 'relax_data.read()' does not read in any relaxation data if
the file containing that data contains more than 4 columns!  Subsequently all
residues are deselected during optimisation, either the 'grid_search' or
'minimise' user functions.  This then causes various failures as not a single
residue is selected.
