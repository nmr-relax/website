structure.read_pdb

Synopsis

The pdb loading function.

Defaults

structure.read_pdb(self, run=None, file=None, dir=None, model=None, load_seq=1)

Keyword Arguments

run: The run to assign the structure to.

file: The name of the PDB file.

dir: The directory where the file is located.

model: The PDB model number.

load_seq: A flag specifying whether the sequence should be loaded from the PDB file.

Description

To load a specific model from the PDB file, set the model flag to an integer . The structure beginning with the line `MODEL ' in the PDB file will be loaded. Otherwise all structures will be loaded starting from the model number 1.

To load the sequence from the PDB file, set the `load_seq' flag to 1. If the sequence has previously been loaded, then this flag will be ignored.

Example

To load all structures from the PDB file `test.pdb' in the directory `~/pdb' for use in the model-free analysis run `m8', type:


relax> structure.read_pdb(`m8', `test.pdb', `~/pdb', 1)


relax> structure.read_pdb(run=`m8', file=`test.pdb', dir=`pdb', model=1)

To load the 10 model from the file `test.pdb', use:


relax> structure.read_pdb(`m1', `test.pdb', model=10)


relax> structure.read_pdb(run=`m1', file=`test.pdb', model=10)
