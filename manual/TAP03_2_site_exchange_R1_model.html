The TAP03 2-site exchange R1 model

This is the model for 2-site exchange for off-resonance R1-type data from Trott et al. (2003). It is selected by setting the model to `TAP03'. The equation for the exchange process is

R1 = R1cos2 + R1'sin2    
  (theparentequation.64)

The equation is accurate when populations are less skewed than the `TP02' model ( pA pB). Note that this model, as with the `TP02' model, has been superseded by the `MP05' model in the next section. The reference for this equation is:

  • Trott03

More information about the TAP03 model is available from:

  • the relax wiki at http://wiki.nmr-relax.com/TAP03,
  • the API documentation at http://www.nmr-relax.com/api/3.1/lib.dispersion.tap03-module.html,
  • the relaxation dispersion page of the relax website at http://www.nmr-relax.com/analyses/relaxation_dispersion.html#TAP03.
