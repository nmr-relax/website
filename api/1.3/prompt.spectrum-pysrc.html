103          replicated.

108   
109          ____________________________________________________________________________________________ 
110          |          |                                        |                                      | 
111          | Int type | Noise source                           | Error scope                          | 
112          |__________|________________________________________|______________________________________| 
113          |          |                                        |                                      | 
114          | Heights  | RMSD baseplane                         | One sigma per peak per spectrum      | 
115          |          |                                        |                                      | 
116          | Heights  | Partial duplicate + variance averaging | One sigma for all peaks, all spectra | 
117          |          |                                        |                                      | 
118          | Heights  | All replicated + variance averaging    | One sigma per replicated spectra set | 
119          |          |                                        |                                      | 
120          | Volumes  | RMSD baseplane                         | One sigma per peak per spectrum      | 
121          |          |                                        |                                      | 
122          | Volumes  | Partial duplicate + variance averaging | One sigma for all peaks, all spectra | 
123          |          |                                        |                                      | 
124          | Volumes  | All replicated + variance averaging    | One sigma per replicated spectra set | 
125          |__________|________________________________________|______________________________________| 
126   
127   
128          Peak heights with baseplane noise RMSD 
129          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
130   
131          When none of the spectra have been replicated, then the peak height errors are calculated 
132          using the RMSD of the baseplane noise, the value of which is set by the 
133          spectrum.baseplane_rmsd() user function.  This results in a different error per peak per 
134          spectrum.  The standard deviation error measure for the peak height, sigma_I, is set to the 
135          RMSD value. 
136   
137   
138          Peak heights with partially replicated spectra 
139          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
140   
141          When spectra are replicated, the variance for a single spin at a single replicated spectra 
142          set is calculated by the formula 
143   
144          ----- 
145   
146              sigma^2 =  sum({Ii - Iav}^2) / (n - 1) , 
147   
148          ----- 
149   
150          where sigma^2 is the variance, sigma is the standard deviation, n is the size of the 
151          replicated spectra set with i being the corresponding index, Ii is the peak intensity for 
152          spectrum i, and Iav is the mean over all spectra i.e. the sum of all peak intensities 
153          divided by n. 
154   
155          As the value of n in the above equation is always very low since normally only a couple of 
156          spectra are collected per replicated spectra set, the variance of all spins is averaged for 
157          a single replicated spectra set.  Although this results in all spins having the same error, 
158          the accuracy of the error estimate is significantly improved. 
159   
160          If there are in addition to the replicated spectra loaded peak intensities which only 
161          consist of a single spectrum, i.e. not all spectra are replicated, then the variances of 
162          replicated replicated spectra sets will be averaged.  This will be used for the entire 
163          experiment so that there will be only a single error value for all spins and for all 
164          spectra. 
165   
166   
167          Peak heights with all spectra replicated 
168          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
169   
170          If all spectra are collected in duplicate (triplicate or higher number of spectra are 
171          supported), the each replicated spectra set will have its own error estimate.  The error 
172          for a single peak is calculated as when partially replicated spectra are collected, and 
173          these are again averaged to give a single error per replicated spectra set.  However as all 
174          replicated spectra sets will have their own error estimate, variance averaging across all 
175          spectra sets will not be performed. 
176   
177   
178          Peak volumes with baseplane noise RMSD 
179          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
180   
181          The method of error analysis when no spectra have been replicated and peak volumes are used 
182          is highly dependent on the integration method.  Many methods simply sum the number of points 
183          within a fixed region, either a box or oval object.  The number of points used, N, must be 
184          specified by another user function in this class.  Then the error is simply given by the sum 
185          of variances: 
186   
187          ----- 
188   
189              sigma_vol^2 = sigma_i^2 * N, 
190   
191          ----- 
192   
193          where sigma_vol is the standard deviation of the volume, sigma_i is the standard deviation 
194          of a single point assumed to be equal to the RMSD of the baseplane noise, and N is the total 
195          number of points used in the summation integration method.  For a box integration method, 
196          this converts to the Nicholson, Kay, Baldisseri, Arango, Young, Bax, and Torchia (1992) 
197          Biochemistry, 31: 5253-5263 equation: 
198   
199          ----- 
200   
201              sigma_vol = sigma_i * sqrt(n*m), 
202   
203          ----- 
204   
205          where n and m are the dimensions of the box.  Note that a number of programs, for example 
206          peakint (http://hugin.ethz.ch/wuthrich/software/xeasy/xeasy_m15.html) does not use all 
207          points within the box.  And if the number N can not be determined, this category of error 
208          analysis is not possible. 
209   
210          Also note that non-point summation methods, for example when line shape fitting is used to 
211          determine peak volumes, the equations above cannot be used.  Hence again this category of 
212          error analysis cannot be used.  This is the case for one of the three integration methods 
213          used by Sparky (http://www.cgl.ucsf.edu/home/sparky/manual/peaks.html#Integration).  And 
214          if fancy techniques are used, for example as Cara does to deconvolute overlapping peaks 
215          (http://www.cara.ethz.ch/Wiki/Integration), this again makes this error analysis impossible. 
216
