Computation time and the numerical integration of the PCS

The numerical integration of the PCS using standard quadratic integration for the frame order models is impractical and cannot be used. A rough estimate for the computation time required for a full analysis of all the frame order models is in the order of 1e7 years. Therefore a number of tricks have been implemented to speed up the calculations.
