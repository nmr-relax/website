Perform the full model-free analysis protocol of d'Auvergne and Gooley, 2008b.

Parameters:
  • save_dir (str) - The directory, where files are saved in.
  • diff_model (str) - The global diffusion model to optimise. This can be one of 'local_tm', 'sphere', 'oblate', 'prolate', 'ellipsoid', or 'final'.
  • mf_models (list of str) - The model-free models.
  • local_tm_models (list of str) - The model-free models.
  • pdb_file (None or str) - The PDB file (set this to None if no structure is available).
  • seq_args (list of lists of [str, None or str, None or int, None or int, None or int, None or int, None or int, None or int, None or int, None or str]) - The sequence data (file name, dir, mol_name_col, res_num_col, res_name_col, spin_num_col, spin_name_col, sep). These are the arguments to the sequence.read() user function, for more information please see the documentation for that function.
  • het_name (None or str) - The heteronucleus atom name corresponding to that of the PDB file (used if the spin name is not in the sequence data).
  • attached_name (str) - The name of the proton attached to the heteronucleus.
  • relax_data (list of lists of [str, str, float, str, None or str, None or int, None or int, None or int, None or int, None or int, None or int, None or int, None or str]) - The relaxation data (data type, frequency label, frequency, file name, dir, mol_name_col, res_num_col, res_name_col, spin_num_col, spin_name_col, data_col, error_col, sep). These are the arguments to the relax_data.read() user function, please see the documentation for that function for more information.
  • unres (None or str) - The file containing the list of unresolved spins to exclude from the analysis (set this to None if no spin is to be excluded).
  • exclude (None or str) - A file containing a list of spins which can be dynamically excluded at any point within the analysis (when set to None, this variable is not used).
  • bond_length (float) - The bond length in metres.
  • csa (float) - The chemical shift anisotropy value (multiply by 1e-6 to convert out of ppm).
  • hetnuc (str) - The heteronucleus type, i.e. '15N', '13C', etc.
  • proton (str) - The proton type, i.e. '1H'.
  • grid_inc (int) - The grid search size (the number of increments per dimension).
  • min_algor (str) - The minimisation algorithm (in most cases this should not be changed).
  • mc_num (int) - The number of Monte Carlo simulations to be used for error analysis at the end of the analysis.
  • max_iter (int or None.) - The maximum number of iterations for the global iteration. Set to None, then the algorithm iterates until convergence.
  • user_fns (dict) - A dictionary of replacement user functions. These will overwrite the standard user functions. The key should be the name of the user function or user function class and the value should be the function or class instance.
  • conv_loop (bool) - Automatic looping over all rounds until convergence.
