Details:

When launching the sequence editor window from a sequence GUI element button
in the user function windows, the user function window will then be hidden
behind the main relax window.  The following steps reproduce the problem:

1.  Launch relax in GUI mode.
2.  Start a new NOE analysis.
3.  Click on the 'View->Spin viewer' menu item.
4.  Click on the 'Load spins' button.
5.  Select 'From a new PDB file' and click on 'Next'.
6.  Click on the sequence editor button on the far right of 'The read model'
field.
7.  Close the sequence editor window.

This will cause the main relax window to come to the front!
