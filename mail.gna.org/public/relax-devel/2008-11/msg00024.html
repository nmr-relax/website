Posted by Edward d'Auvergne on November 21, 2008 - 13:33:
Hi,

This is just a quick message to explain the recently created
'j_couplings' branch.  This new branch is for the development of code
to handle dynamically averaged J-couplings which will initially be
incorporated into the N-state model analysis which is currently being
developed within relax.  This code will be created by a PhD student
Han Sun (hasu att nmr dot mpibpc dot mpg dot de), developed under my
supervision here at the MPI in Goettingen, Germany.  Note that the
normal open source processes will be followed with her submitting
patches to relax-devel which I will then apply and commit (assuming
the code is good), followed later on by the normal voting procedure
for developer inclusion (again assuming the code is good ;).  With
relax being able to handle dynamically averaged J-couplings, this
should open future avenues for new types of dynamics analyses.

Regards,

Edward
