 30      """Calculate the rotation matrix required to rotate from one vector to another. 
 31   
 32      For the rotation of one vector to another, there are an infinit series of rotation matrices 
 33      possible.  Due to axially symmetry, the rotation axis can be any vector lying in the symmetry 
 34      plane between the two vectors.  Hence the axis-angle convention will be used to construct the 
 35      matrix with the rotation axis defined as the cross product of the two vectors.  The rotation 
 36      angle is the arccosine of the dot product of the two unit vectors. 
 37   
 38      Given a unit vector parallel to the rotation axis, w = [x, y, z] and the rotation angle a, 
 39      the rotation matrix R is: 
 40   
 41                |  1 + (1-cos(a))*(x*x-1)   -z*sin(a)+(1-cos(a))*x*y   y*sin(a)+(1-cos(a))*x*z | 
 42          R  =  |  z*sin(a)+(1-cos(a))*x*y   1 + (1-cos(a))*(y*y-1)   -x*sin(a)+(1-cos(a))*y*z | 
 43                | -y*sin(a)+(1-cos(a))*x*z   x*sin(a)+(1-cos(a))*y*z   1 + (1-cos(a))*(z*z-1)  | 
 44   
 45      @param R:           The 3x3 rotation matrix to update. 
 46      @type R:            3x3 numpy array 
 47      @param vector_orig: The unrotated vector defined in the reference frame. 
 48      @type vector_orig:  numpy array, len 3 
 49      @param vector_fin:  The rotated vector defined in the reference frame. 
 50      @type vector_fin:   numpy array, len 3 
 51      """ 
 52   
 53      # Convert the vectors to unit vectors. 
 54      vector_orig = vector_orig / norm(vector_orig) 
 55      vector_fin = vector_fin / norm(vector_fin) 
 56   
 57      # The rotation axis. 
 58      axis = cross(vector_orig, vector_fin) 
 59      x = axis[0] 
 60      y = axis[1] 
 61      z = axis[2] 
 62   
 63      # The rotation angle. 
 64      angle = acos(dot(vector_orig, vector_fin)) 
 65   
 66      # Trig functions (only need to do this maths once!). 
 67      ca = cos(angle) 
 68      sa = sin(angle) 
 69   
 70      # Calculate the rotation matrix elements. 
 71      R[0,0] = 1.0 + (1.0 - ca)*(x**2 - 1.0) 
 72      R[0,1] = -z*sa + (1.0 - ca)*x*y 
 73      R[0,2] = y*sa + (1.0 - ca)*x*z 
 74      R[1,0] = z*sa+(1.0 - ca)*x*y 
 75      R[1,1] = 1.0 + (1.0 - ca)*(y**2 - 1.0) 
 76      R[1,2] = -x*sa+(1.0 - ca)*y*z 
 77      R[2,0] = -y*sa+(1.0 - ca)*x*z 
 78      R[2,1] = x*sa+(1.0 - ca)*y*z 
 79      R[2,2] = 1.0 + (1.0 - ca)*(z**2 - 1.0)
