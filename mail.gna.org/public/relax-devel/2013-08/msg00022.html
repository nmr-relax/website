Follow-up Comment #1:

You need to run 'scons clean' first to remove the *.pyc files and any other
temporary files.

The "No documentation available!" errors are normal as these are for
non-python files which are not part of relax.  It's safe to ignore these.

The wxPython indentation errors are also fine.  I don't know why epydoc is
scanning the wx module docstrings, but it cannot be stopped.

The "Unable to render Graphviz dot graph" are because the Graphviz software is
not available.  Windows binaries are available though
(http://www.graphviz.org).

Closing as 'Wont fix' as there is nothing to be fixed.
