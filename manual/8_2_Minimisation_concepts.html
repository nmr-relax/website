8.2 Minimisation concepts

8.2.1 The function value

At the simplest level all minimisation techniques require at least a function which will supply a single value for different parameter values . For the modelling of NMR relaxation data this function is the chi-squared equation (6.1) on page . For certain algorithms, such a simplex minimisation, this single value suffices.
