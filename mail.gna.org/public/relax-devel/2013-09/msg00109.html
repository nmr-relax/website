Posted by Troels Emtekær Linnet on September 10, 2013 - 09:00:
You can close it.

It is a rather specific re-quest, and is most easily solved by an
external script.
http://wiki.nmr-relax.com/Tutorial_for_Relaxation_dispersion_analysis_cpmg_fixed_time_recorded_on_varian_as_fid_interleaved_scripts#stPeakList.pl

It would need that relax should point to and .ft2 file, and read the
header of this file or make the user make the input of:
OBS MHz
DATA SIZE
ORIG Hz
SW Hz

together with a file with ppm values for the two dimensions.

And thereafter use NMRPipe seriestab.
And it would not fit on a windows computer.

So all in all, before this feature gets more "popular", I think it
should be closed.
