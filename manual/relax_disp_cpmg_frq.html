relax_disp.cpmg_frq

Synopsis

Set the CPMG frequency associated with a given spectrum.

Defaults

relax_disp.cpmg_frq(spectrum_id=None, cpmg_frq=None)

Keyword arguments

spectrum_id: The spectrum ID string to associate the CPMG frequency to.

cpmg_frq: The frequency, in Hz, of the CPMG pulse train.

Description

This allows the CPMG pulse train frequency of a given spectrum to be set. If None is given for frequency, then the spectrum will be treated as a reference spectrum.

Prompt examples

To identify the reference spectrum called `reference_spectrum', type one of: 

[numbers=none]
relax> relax_disp.cpmg_frq(None, 'reference_spectrum')


[numbers=none]
relax> relax_disp.cpmg_frq(cpmg_frq=None, spectrum_id='reference_spectrum')

To set a frequency of 200 Hz for the spectrum `200_Hz_spectrum', type one of: 

[numbers=none]
relax> relax_disp.cpmg_frq(200, '200_Hz_spectrum')


[numbers=none]
relax> relax_disp.cpmg_frq(cpmg_frq=200, spectrum_id='200_Hz_spectrum')
