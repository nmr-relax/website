About relax-users

This is the relax-users mailing list for the relax NMR software project. It is a list for general discussions about the program and for asking questions about the usage of the software. To see the collection of prior postings to the list, visit the relax-users Archives.

Gna! shutdown
The Gna! open source infrastructure was shut down at the start of 2017. Hence the relax-users@gna.org mailing list no longer exists. This page is a copy of the original, modified for archival purposes. It was taken from the Internet Archive.
