Posted by Edward d'Auvergne on April 04, 2008 - 10:05:
Hi,

The 1.2 line user function:

relax> run.create(name, 'mf')

has been renamed in the 1.3 line to:

relax> pipe.create(name, 'mf')

Although the two user functions appear similar, their effect on the
relax data storage object is quite different (because the 1.2 and 1.3
line data stores are completely different).

Regards,

Edward
