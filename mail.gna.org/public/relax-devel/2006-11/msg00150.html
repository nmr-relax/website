Details:

The Brownian rotational diffusion equations currently in the model-free
chapter of the relax user manual should be shifted into its own chapter
preceding the model-free chapter.  Once the 'tensor_pdb' branch has been
merged and the 'pdb' user function class renamed to 'structure, the
'structure.create_tensor_pdb()' and 'pymol.tensor_pdb()' user functions
should be described in that chapter and pictures of the resultant tensors
added.
