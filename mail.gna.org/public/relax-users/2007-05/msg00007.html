Posted by Edward d'Auvergne on May 08, 2007 - 09:34:
I think the problem may be that not enough iterations of the sphere,
prolate and oblate spheroids, and the ellipsoid have been carried out.
To run the script, one iteration with self.diff_model='local_tm' is
initially required.  Then for the sphere, you will need to run the
script numerous times (maybe 5 or more iterations) until the script
converges.  I would recommend using relax-1.2.10 as the full analysis
script will tell you when convergence has occurred.  Multiple iterations
of the prolate spheroid, oblate spheroid, and ellipsoid are separately
required (possibly running the script between 10 and 15 time).  Once
each of these four diffusion models have converged, then the final run
is executed.  I hope this helps.

Regards,

Edward
