Perform relaxation curve fitting.

Parameters:
  • pipe_name (str) - The name of the data pipe to create.
  • noe_ref (file) - The NOE reference peak file.
  • noe_ref_rmsd (int) - Background RMSD of reference NOE spectrum.
  • sat_ref (file) - The NOE saturated peak file.
  • noe_sat_rmsd (int) - Background RMSD of saturated NOE spectrum.
  • seq_args (list of lists of [str, None or str, None or int, None or int, None or int, None or int, None or int, None or int, None or int, None or str]) - The sequence data (file name, dir, mol_name_col, res_num_col, res_name_col, spin_num_col, spin_name_col, sep). These are the arguments to the sequence.read() user function, for more information please see the documentation for that function.
  • unresolved (str) - Residues to exclude.
  • pdb_file (str) - Structure file in pdb format.
  • output_file (str) - Name of the output file.
  • results_dir (str) - Folder where results files are placed in.
  • int_method (str) - The integration method, one of 'height', 'point sum' or 'other'.
  • heteronuc (str) - Name of heteronucleus of peak list.
  • proton (str) - Name of proton of peak list.
  • heteronuc_pdb (str) - Name of heteronucleus of PDB file.
