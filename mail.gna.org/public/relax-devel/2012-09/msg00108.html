This bug is invalid.  Art Palmer's Modelfree4 program does not and will never,
ever run on MS Windows!  If unsure, please ask questions on the relax-users
mailing list before submitting bug reports.  Also, before submitting a bug
report, please read the documentation for the user function where the error
has occurred.  Note that RelaxErrors are not bugs.  A RelaxError indicates
that the fault is on the side of the user, and the error message itself is
telling you what the problem is.
