This is a warning about the file 'specific_fns/model_free.py'.  As the
file is heading towards 6000 lines with all functions contained within
a single class, the file is getting to big to handle.  Therefore I
will try to group related functions into their own classes within that
file.  For example, all the Molmol functions will be shifted to the
'Molmol' class located after the main 'Model_free' class.  To access
the functions, type 'self.relax.specific.model_free.molmol.macro',
etc.  As these are large rearrangments, any modifications made to
'specific_fns/model_free.py' could end up being very difficult to
merge with these changes.
