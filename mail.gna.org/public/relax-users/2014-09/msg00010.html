Dear all,

I have noticed that a number of new subscribers to the relax-users
mailing list are missing out on the relax announcements.  These are
only available via the relax-announce mailing list.  If you do not
receive these messages, please subscribe to the mailing list at
https://mail.gna.org/listinfo/relax-announce.  You will only receive
~10 messages per year from this list (see
http://news.gmane.org/gmane.science.nmr.relax.announce).

Cheers,

Edward
