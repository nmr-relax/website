+            Da: 1e5
+            theta: 0 degrees

+
+        These MC sim values will then be explicity overwritten by setting

+            tm: 0.5 ns
+            Da: 3e5
+            theta: 2 degrees
+            phi: 0 degrees

+        # Set the diffusion type.
