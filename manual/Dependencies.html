Dependencies

The following packages need to be installed before using relax:

http://python.org/Python:
Version 2.5 or higher.
http://numpy.scipy.org/NumPy:
Version 1.0.4 or higher. This package is used for most of the numerical calculations within relax.
http://www.scipy.org/SciPy:
Version 0.7.1 or higher. This package is optional. It is required only for the frame order theory analyses.
http://www.scipy.org/wxPython:
Version 2.8 or higher. This package is also optional. It is required for the operation of the graphical user interface (GUI).
http://mpi4py.scipy.org/mpi4py:
Version 1.2 or higher. This optional dependency is essential for running relax in MPI multi-processor mode.

Older versions of these packages may work, use them at your own risk. If, for older dependency versions, errors do occur please submit a bug report to the bug tracker at https://gna.org/bugs/?group=relaxhttps://gna.org/bugs/?group=relax. That way a solution may be created for the next relax release.

Note that only the official Python distribution from http://python.orghttp://python.org is supported. If you use the Enthought Python Distribution (EPD) or other non-official distributions you may encounter problems with the relax C modules, the graphical user interface, or other issues. These alternative distributions are to be used at your own risk. Any issues encountered will not be considered as relax bugs.
