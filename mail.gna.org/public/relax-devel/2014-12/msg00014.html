Posted by Edward d'Auvergne on December 03, 2014 - 22:23:
Hi Troels,

For the wizards, the graphic needs to be 200 px wide.  Do you have a
copy of the oxygen-icon repository?  If not, check it out with
subversion with the command:

$ svn co svn://anonsvn.kde.org/home/kde/trunk/kdesupport/oxygen-icons

Then find the document-preview.svgz file.  You can then output this to
200x200 pixels, and replace this graphic.  The algorithm for laying
out the wizard GUI elements depends on this size, so strange things
might happen if it is too small.  You may like to also choose and
Oxygen icon for the io user function class, as it currently shows the
relax butterfly in the menus.  If you have the oxygen-icon repository,
feel free to add any to relax, but remember that it needs to be there
in all sizes.  For example the HTML manual absolutely requires the
128x128 versions.  And the menus require the 16x16 versions.  Buttons
require the 22x22, 32x32, or 48x48 versions.

Regards,

Edward
