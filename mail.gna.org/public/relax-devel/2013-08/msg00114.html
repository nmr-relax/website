Posted by Edward d'Auvergne on August 20, 2013 - 23:12:
Hi,

I'm not sure how well the kA parameter is currently supported in
relax.  You may encounter some problems somewhere.  In any case, do
you know what happened to the pA and pB parameters in this model?  Is
pA ~ 1 so that it does not appear in the equation?  The pA parameter
is needed to convert between kA and kex, which will need to be
performed either in the lib.dispersion code if kex is the model
parameter, or at the end of the analysis specific minimise() function
to store the kex value.

Regards,

Edward
