relax_data(self, file)

source code  
Open the relaxation data in the file 'file' and return all the data.

It is assumed that the file has four columns separated by whitespace.  The columns should be:
        0 - Residue numbers
        1 - Residue names
        2 - R1, R2, or NOE values
        3 - The errors
