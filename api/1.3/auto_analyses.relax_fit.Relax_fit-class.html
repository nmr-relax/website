Perform relaxation curve fitting.

Parameters:
  • file_root (str) - File root of the output filea.
  • pipe_name (str) - The name of the data pipe to create.
  • results_directory (str) - The directory, where results files are saved.
  • seq_args (list of lists of [str, None or str, None or int, None or int, None or int, None or int, None or int, None or int, None or int, None or str]) - The sequence data (file name, dir, mol_name_col, res_num_col, res_name_col, spin_num_col, spin_name_col, sep). These are the arguments to the sequence.read() user function, for more information please see the documentation for that function.
  • file_names (list of str) - A list of all the peak list file names.
  • relax_times (list of float) - The list of relaxation times corresponding to file_names. These two lists must be of the same size.
  • pdb_file (str) - Name of the PDB file.
  • unresolved (str) - Filename of the unresolved spin list.
  • int_method (str) - The integration method, one of 'height', 'point sum' or 'other'.
  • heteronuc (str) - Label of hetero nucleus in the peak intensity file.
  • proton (str) - Label of proton in the peak intensity file.
  • load_spin_ids (str) - Id of spin to extract in PDB file.
  • mc_num (int) - The number of Monte Carlo simulations to be used for error analysis at the end of the analysis.
  • inc (int) - Number of grid search increments.
  • view_plots (boolean) - Flag to automatically view grace plots after calculation.
