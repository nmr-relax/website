Hi Michael,

I've now added icons for all the menu sub items, and cleaned up that
part of the code.  What do you think?  Do you have access to the
oxygen icon set?  If not, I've temporarily placed a copy of all these
icons at http://medusa.nmr.mpibpc.mpg.de/~edau/oxygen_icons.tar.bz2

to use for the current menu entries?  I've tried to match KDE and
GNOME apps as much as possible for icon selection.  Note that although
the oxygen icons are used in KDE, they are not part of KDE.
