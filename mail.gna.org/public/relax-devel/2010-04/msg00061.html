Hi,

This BIC model select can be reintroduced in the future if someone
implements another type of auto-analysis which performs the original
initial diffusion tensor estimate analysis (which dauvergne_protocol
is designed to replace).  The functionality will always be there in
relax.  So if someone designs a new analysis type around the Bayesian
philosophy underlying BIC - i.e. uses priors to help solve this
problem - then BIC can be used in the GUI.
