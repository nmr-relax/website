Hi Kozo,

This warning message is perfectly normal.  The colour is actually dark
orange, red is used for the RelaxError messages (you can see this in
the gui/controller.py file on line 616).  The protons are deselected
as I'm assuming that you are looking at nitrogen spin relaxation
rather than proton spin relaxation.  You can remove such warnings by
deselecting the protons prior to optimisation, but this is not needed.
 You should carefully check all other warnings though to be sure that
relax is doing what you would expect.

Regards,

Edward
