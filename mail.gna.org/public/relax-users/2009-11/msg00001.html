This is most definitely a bug in relax!  This doesn't appear to be

display criteria to show all bugs).  Would you be able to create a bug

make, but it would be of significant help if you could provide a
little more information so that I can replicate the bug.  Ideally if
you could create a tiny data set of 1 or 2 residues which replicates

problem into the test suite).  The data can be randomised if you
prefer that no one sees it yet.
