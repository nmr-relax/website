+For all of these files, the experiment type is SQ CPMG and the relaxation 
delay time is 30~ms.
+All files are located in the 
\directory{test\osus{}suite\ossep{}shared\osus{}data\ossep{}dispersion\ossep{}Hansen}
 directory.
+
+To simplify the loading of the data, all of the 500 MHz data will be read 
simultaneously (followed by all of the 800 MHz data).
+Firstly, to select all of the 500~MHz files, click on the button at the end 
of the \guistring{file name(s)} GUI element.
+This will launch the multiple file selection window:
+
+\begin{minipage}[h]{\linewidth}
+  \centerline{
+    \includegraphics[
+      width=0.9\textwidth,
+      bb=14 14 1415 1065

+  }
+\end{minipage}
+
+Add the first file by typing the file name or clicking on the file selection 
button to the right of the file input field and choosing the file.
+A preview button is included to allow the file to be checked.
+Then click on the \guibutton{Add} button to insert a new file selection 
item, add the next file, and continue until complete.
+You should now see:
+
+\begin{minipage}[h]{\linewidth}
+  \centerline{
+    \includegraphics[
+      width=0.9\textwidth,

+Click on \guibutton{OK} to return to the 
\uf{spectrum\ufsep{}read\ufus{}intensities} user function wizard page.
+Next, the spectrum IDs should be set.
+For this, click on the button to the far right of the spectrum ID string 
element.
+Click on \guibutton{Add} and fill in the spectrum IDs corresponding to the 
files selected.
+As 19 files exist for the 500 MHz data, 19 spectrum IDs should be added.
+You will now see:
+
+\begin{minipage}[h]{\linewidth}
+  \centerline{
+    \includegraphics[

+Click on \guibutton{OK} to return to the 
\uf{spectrum\ufsep{}read\ufus{}intensities} user function wizard page.
