OpenDX

Version 4.1.3 or compatible. OpenDX is used for viewing the output of the space mapping function and is executed by passing the command dx to the command line with various options. The program is designed for visualising multidimensional data and can be found at http://www.opendx.org/http://www.opendx.org/.
