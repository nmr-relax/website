Introduction to frame order data analysis

The analysis consists of two parts. The first is the single structure or ensemble analysis of the combined RDC and PCS alignment tensor for the non-moving paramagnetically aligned domain or rigid body. The resultant alignment tensor and optimised paramagnetic Ln3+ position is then used as input into the frame order analysis.
