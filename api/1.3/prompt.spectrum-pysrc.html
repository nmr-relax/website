 80          This user function must only be called after all peak intensities have been loaded and all 
 81          other necessary spectral information set.  This includes the baseplane RMSD and the number 
 82          of points used in volume integration, both of which are only used if spectra have not been 
 83          replicated. 
 84   
 85          Six different types of error analysis are supported depending on whether peak heights or 
 86          volumes are supplied, whether noise is determined from replicated spectra or the RMSD of the 
 87          baseplane noise, and whether all spectra or only a subset have been duplicated.  These are: 
 88   
 89          ____________________________________________________________________________________________ 
 90          |          |                                        |                                      | 
 91          | Int type | Noise source                           | Error scope                          | 
 92          |__________|________________________________________|______________________________________| 
 93          |          |                                        |                                      | 
 94          | Heights  | RMSD baseplane                         | One sigma per peak per spectrum      | 
 95          |          |                                        |                                      | 
 96          | Heights  | Partial duplicate + variance averaging | One sigma for all peaks, all spectra | 
 97          |          |                                        |                                      | 
 98          | Heights  | All replicated + variance averaging    | One sigma per replicated spectra set | 
 99          |          |                                        |                                      | 
100          | Volumes  | RMSD baseplane                         | One sigma per peak per spectrum      | 
101          |          |                                        |                                      | 
102          | Volumes  | Partial duplicate + variance averaging | One sigma for all peaks, all spectra | 
103          |          |                                        |                                      | 
104          | Volumes  | All replicated + variance averaging    | One sigma per replicated spectra set | 
105          |__________|________________________________________|______________________________________| 
106   
107   
108          Peak heights with baseplane noise RMSD 
109          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
110   
111          When none of the spectra have been replicated, then the peak height errors are calculated 
112          using the RMSD of the baseplane noise, the value of which is set by the 
113          spectrum.baseplane_rmsd() user function.  This results in a different error per peak per 
114          spectrum.  The standard deviation error measure for the peak height, sigma_I, is set to the 
115          RMSD value. 
116   
117   
118          Peak heights with partially replicated spectra 
119          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
120   
121          When spectra are replicated, the variance for a single spin at a single replicated spectra 
122          set is calculated by the formula 
123   
124          ----- 
125   
126              sigma^2 =  sum({Ii - Iav}^2) / (n - 1) , 
127   
128          ----- 
129   
130          where sigma^2 is the variance, sigma is the standard deviation, n is the size of the 
131          replicated spectra set with i being the corresponding index, Ii is the peak intensity for 
132          spectrum i, and Iav is the mean over all spectra i.e. the sum of all peak intensities 
133          divided by n. 
134   
135          As the value of n in the above equation is always very low since normally only a couple of 
136          spectra are collected per replicated spectra set, the variance of all spins is averaged for 
137          a single replicated spectra set.  Although this results in all spins having the same error, 
138          the accuracy of the error estimate is significantly improved. 
139   
140          If there are in addition to the replicated spectra loaded peak intensities which only 
141          consist of a single spectrum, i.e. not all spectra are replicated, then the variances of 
142          replicated replicated spectra sets will be averaged.  This will be used for the entire 
143          experiment so that there will be only a single error value for all spins and for all 
144          spectra. 
145   
146   
147          Peak heights with all spectra replicated 
148          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
149   
150          If all spectra are collected in duplicate (triplicate or higher number of spectra are 
151          supported), the each replicated spectra set will have its own error estimate.  The error 
152          for a single peak is calculated as when partially replicated spectra are collected, and 
153          these are again averaged to give a single error per replicated spectra set.  However as all 
154          replicated spectra sets will have their own error estimate, variance averaging across all 
155          spectra sets will not be performed. 
156   
157   
158          Peak volumes with baseplane noise RMSD 
159          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
160   
161          The method of error analysis when no spectra have been replicated and peak volumes are used 
162          is highly dependent on the integration method.  Many methods simply sum the number of points 
163          within a fixed region, either a box or oval object.  The number of points used, N, must be 
164          specified by another user function in this class.  Then the error is simply given by the sum 
165          of variances: 
166   
167          ----- 
168   
169              sigma_vol^2 = sigma_i^2 * N, 
170   
171          ----- 
172   
173          where sigma_vol is the standard deviation of the volume, sigma_i is the standard deviation 
174          of a single point assumed to be equal to the RMSD of the baseplane noise, and N is the total 
175          number of points used in the summation integration method.  For a box integration method, 
176          this converts to the Nicholson, Kay, Baldisseri, Arango, Young, Bax, and Torchia (1992) 
177          Biochemistry, 31: 5253-5263 equation: 
178   
179          ----- 
180   
181              sigma_vol = sigma_i * sqrt(n*m), 
182   
183          ----- 
184   
185          where n and m are the dimensions of the box.  Note that a number of programs, for example 
186          peakint (http://hugin.ethz.ch/wuthrich/software/xeasy/xeasy_m15.html) does not use all 
187          points within the box.  And if the number N can not be determined, this category of error 
188          analysis is not possible. 
189   
190          Also note that non-point summation methods, for example when line shape fitting is used to 
191          determine peak volumes, the equations above cannot be used.  Hence again this category of 
192          error analysis cannot be used.  This is the case for one of the three integration methods 
193          used by Sparky (http://www.cgl.ucsf.edu/home/sparky/manual/peaks.html#Integration).  And 
194          if fancy techniques are used, for example as Cara does to deconvolute overlapping peaks 
195          (http://www.cara.ethz.ch/Wiki/Integration), this again makes this error analysis impossible. 
196   
197   
198          Peak volumes with partially replicated spectra 
199          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
200   
201          When peak volumes are measured by any integration method and a few of the spectra are 
202          replicated, then the intensity errors are calculated identically as described in the 'Peak 
203          heights with partially replicated spectra' section above. 
204   
205   
206          Peak volumes with all spectra replicated 
207          ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 
208   
209          With all spectra replicated and again using any integration methodology, the intensity 
210          errors can be calculated as described in the 'Peak heights with all spectra replicated' 
211          section above. 
212          """
