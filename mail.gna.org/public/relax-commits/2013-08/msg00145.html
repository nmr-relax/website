Log:
Added the new 'TSMFK01' model to the 
specific_analyses\relax_disp\variables.py module.

This is the Tollinger/Kay model for the 2-site very-slow exchange model for 
CPMG-type experiments, which cover the range of microsecond to second time 
scale.
Paper by M. Tollinger, N.R. Skrynnikov, F.A.A. Mulder, J.D.F. Kay and L.E. 
Kay (2001).

Progress sr #3071: https://gna.org/support/index.php?3071 - Implementation of 
Tollinger/Kay dispersion model (2001)
Following the guide at: 
http://wiki.nmr-relax.com/Tutorial_for_adding_relaxation_dispersion_models_to_relax

Troels E. Linnet provided this patch. Commit by: tlinnet _aaattt_ 
gmail_dot_com
