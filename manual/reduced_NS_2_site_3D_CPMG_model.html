The reduced NS 2-site 3D CPMG model

This is the numerical model for 2-site exchange using 3D magnetisation vectors, whereby the simplification R2A0 = R2B0 is assumed. It is selected by setting the model to `NS CPMG 2-site 3D'. The simple constraint pA > pB is used to halve the optimisation space, as both sides of the limit are mirror image spaces.
