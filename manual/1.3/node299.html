pipe.switch()

Synopsis

Function for switching between data pipes.

Defaults

pipe.
[0]switch(
[0]self, pipe_name=
[0]None)

Keyword Arguments

pipe_name: The name of the data pipe.

Description

This function will switch from the current data pipe to the given data pipe.

Examples

To switch to the `ellipsoid' data pipe, type:


relax> pipe.
[0]
[0]switch(
[0]
[0]`ellipsoid')


relax> pipe.
[0]
[0]switch(
[0]
[0]pipe_name=
[0]
[0]`ellipsoid')
