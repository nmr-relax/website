Posted by Edward d'Auvergne on February 24, 2012 - 21:17:
Hi Jimmy,

Welcome to the relax mailing lists.  The error you are seeing seems to
be a system administration issue.  Are you the administrator for the
computer you are using?  Can you copy any random file to C:/Program
Files?  If not, and Python, numpy, and the other dependencies you
would like (such as wxpython for the GUI) are already installed, then
you can run relax from anywhere using the windows powershell or cmd
program.  Note that relax doesn't come with any special installation
binaries (as no relax user has created this yet) so you won't get
program icons on the desktop or in the menus.  If you can copy random
files to C:/Program Files, then we will need to dig a bit further into
the problem.

Regards,

Edward
