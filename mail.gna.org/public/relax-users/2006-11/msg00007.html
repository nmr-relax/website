Hi,

If anyone is interested, I've created flow diagrams detailing a number
of model-free optimisation protocols.  These changes have been made to
the 1.2 and 1.3 repository lines (http://svn.gna.org/viewcvs/relax/),
and the updated PDF user manual can be downloaded directly from the
Subversion repository using the link
http://svn.gna.org/viewcvs/*checkout*/relax/1.2/docs/relax.pdf?rev=2835.

Three diagrams have been added.  The first details the techniques of
Mandel et al., 1995, including the hypothesis testing ANOVA model
selection.  The second is the changes to model-free analysis when AIC
model selection is used for selecting between model-free models as
well as diffusion models and when using model elimination to remove
failed model-free models.  The third diagram describes the new, and as
of yet unpublished, model-free optimisation protocol of the
'full_analysis.py' sample script.  If you have been using this script,
the flow diagram should be helpful for understanding the fine details
of the protocol (and why the approach is very different from anything
currently being used by the field).

Cheers,

Edward
