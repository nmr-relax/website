A number of improvements to the http://nmr-relax.com web pages.

The relax menus on all pages has been expanded.  The page http://nmr-relax.com/features.html now
exists in all the menus.  A new section in the menu called 'Developers & power users' has been
created.  This section contains links to the development platform page
(http://nmr-relax.com/devel_platform.html) and to the API documentation which is not currently in
the CVS repository.

The 'Development platforms' subsection of the 'Source code release (for all other systems)' section
of the download page (http://nmr-relax.com/download.html) has been modified.  This now points to the
development platform page (http://nmr-relax.com/devel_platform.html).

'Modelfree4' has been renamed to 'Modelfree' on the main relax web page
(http://nmr-relax.com/index.html).  This is for Googling reasons!  The text 'GNU GPL (General Public
Licence)' on this page has been changed to 'GNU General Public Licence (GPL)' as well.

The CSS style of the footer has been improved.
