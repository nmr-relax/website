Added the synthetic saturation-recovery data in the form of Sparky peak lists 
to the repository.

These files were created by Andras Boeszoermenyi <Andras_Boeszoermenyi att 
hms dott harvard dot
edu>.  They are attached to the task at http://gna.org/task/?7415 as the 
Relax_sym.tar.gz file at
http://gna.org/task/download.php?file_id=22989.  They were created for the 
formula I0*(1 -
exp(âˆ’R1*t)) where I0 = 1000000000000000.00 and R1 = 0.5.  These files and the 
associated
relax_sim.py script (which needs to be updated for the latest relax version) 
could form the basis of
a basic system test.  This system test could then be used to implement the 
saturation-recovery
experiment equations in relax.
