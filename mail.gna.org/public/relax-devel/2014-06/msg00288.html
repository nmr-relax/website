Details:

The 2 attached scripts demonstrate the problem.  Place these in the base
directory of the relax trunk (r24031 works), then run them as relax scripts. 
The logs have been attached for reference.

As both of these optimisations do not violate any constraints, they should
give very similar results.  They should not be the same, as one uses an
iterative constraint algorithm and the other does not.  But the parameters
should be close to identical.
