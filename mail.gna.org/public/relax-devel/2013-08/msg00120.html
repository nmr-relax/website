Posted by Edward d'Auvergne on August 22, 2013 - 15:05:
Hello,

If you are playing with the relaxation dispersion branch of relax, are
having problems in compiling the C modules using scons, and have not
collected full exponential curve data sets the following commit might
be of interest:

http://article.gmane.org/gmane.science.nmr.relax.scm/18435

It is now possible to continue without the modules being compiled.
This will cause some strange errors in the future when someone does
have full exponential curves, but we can deal with that when the user
reports a problem.  The problem should be rare enough that users
should hopefully not encounter it.

Regards,
