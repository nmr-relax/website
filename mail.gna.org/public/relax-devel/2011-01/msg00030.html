Hi Seb,

It's looking good!  For a list, it might be good to copy what is done
on the features page without (http://www.nmr-relax.com/features.html).
 I.e. remove the 'class="main_ul"' part of the UL element.  Or maybe
it would be better to have <h3> sections for each conference, include
a bit of a description and logo (i.e. 50th Experimental Nuclear
Magnetic Resonance Conference...)?  I'm not really sure though, feel
free do as you wish.  You can even start to play with your own CSS
styling if you like.  If you do, just try to keep to the relaxing blue
theme for relax.  It might also be worth adding the file
enc_2009.tar.bz2 to the repository as well, as a source of all the
graphics and text of the poster, and including a link on the page.  If
that's the case, then we should have a bit of a disclaimer saying that
if someone would like to use it, they must ask the authors for
permission first.  What do you think?
