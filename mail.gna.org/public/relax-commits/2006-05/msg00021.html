Log:
A number of improvements to the manual.

The relaxation curve-fitting chapter now has a mini-title to improve the 
header.

The file 'relax.bst' has been added.  This is a bibliography style file which 
has been modified from
the `apalike' file.  This significantly improves the bibliography layout.

The 'Values, gradients, and Hessians' chapter has been expanded.  A section 
describing the four
parameter combinations has been added.  Another section describing the 
construction of the value,
gradient, and Hessian has been added as well.  This section contains the 
Hessian kite diagram and
the gradient construction diagram.  A number of other fixes have been made to 
the chapter.

The bibtex bibliography file has been significantly expanded for future 
referencing.

Added a forth round of LaTeX compilation to improve the referencing.
