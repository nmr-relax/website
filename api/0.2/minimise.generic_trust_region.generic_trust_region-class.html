calc_rho(self)

source code  
Function to calculate the ratio rho used to choose the trust region radius.

The ratio is defined as:

                f(xk) - f(xk + pk)
        rho  =  ------------------
                  mk(0) - mk(pk)

Where the numerator is called the actual reduction and the denominator is the predicted reduction.
