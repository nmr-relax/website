Hi Mengjun,

For the message:

RelaxError: The keyword argument 'sampleconditions' is unknown.

this is relax telling you directly that you have done something wrong.
 I would recommend you look at the documentation for the bmrb.read
user function (by typing help(bmrb.read) in the relax prompt mode or
looking at the manual http://www.nmr-relax.com/manual/bmrb_read.html).
 From this documentation, you should see why sampleconditions as an
argument is a problem.  I hope this helps.

Regards,

Edward
