Log:
Changed the compression back 9, when creating grace png files.

It doesn't change the quality, just the time to create the file and the size 
of the file.
PNG is lossless, so compression levels 1 to 9 are all pixel-perfect.

The B14 model is explained in: http://wiki.nmr-relax.com/B14.
