About relax-commits

This is the relax-commits mailing list for the relax NMR software project. It is a read only list of all of the commits to the relax source code repository. To see the collection of prior postings to the list, visit the relax-commits Archives.

Gna! shutdown
The Gna! open source infrastructure was shut down at the start of 2017. Hence the relax-commits@gna.org mailing list no longer exists. This page is a copy of the original, modified for archival purposes. It was taken from the Internet Archive.
