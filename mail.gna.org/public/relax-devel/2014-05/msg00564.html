On my test machine, a different behaviour is seen.  No message is printed, but
instead the GUI freezes.  The steps to reproduce are:

1)  Launch the GUI.
2)  Open the dx.map user function via the user function menus.
3)  Click on the button next to the 'lower bounds', 'upper bounds', or 'the
highlight points in the space' elements to open the Sequence or Sequence_2D
windows.
4)  Close the Sequence or Sequence_2D window that appears.

The GUI will now be in a broken state.
