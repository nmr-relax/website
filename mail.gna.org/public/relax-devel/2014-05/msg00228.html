Details:

The dx.map can accept a point [x, y, x]
to map on the chi2 surface.

dx.map(params=dx_params, map_type='Iso3D', spin_id=cur_spin_id, inc=ds.dx_inc,
lower=None, upper=None, axis_incs=10, file_prefix=file_name_map,
dir=ds.resdir, point=dx_set_val, point_file=file_name_point, remap=None)

This should be expanded to:
[[x1, y1, z1], [x2, y2, z2]]

This is to allow, that one can have dynamic parameters points, that showed how
data is generated, and a point which shows which point space was minimised to.
