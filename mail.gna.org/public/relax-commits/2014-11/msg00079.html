Log:
The 1st angle for the calculated frame order matrix graphs is 0 for all non 
pseudo-ellipse models.

This is for the frame_order_solution.py script.  Only the pseudo-ellipse 
models where numerical
integration is required fail for the angle of 0.0.  Therefore the changing of 
the first angle from
0.0 to 0.01 only occurs for the pseudo-ellipse models.  All graphs have been 
updated.
