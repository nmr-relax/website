Hi Troels,

Note that after this commit, you must absolutely use the new minfx
1.0.7 that I just released
(https://gna.org/forum/forum.php?forum_id=2466).  All relaxation
dispersion system tests now pass in trunk, including the new
Relax_disp.test_lm63_3site_synthetic test.  You should restore the
Relax_disp.test_lm63_3site_synthetic system test you deleted and
adjust the values so that we have better test coverage of the 'LM63
3-site' model (http://wiki.nmr-relax.com/LM63_3-site) as mentioned in
http://thread.gmane.org/gmane.science.nmr.relax.scm/21774/focus=6300.

Regards,

Edward
