bmrb.display

Synopsis

Display the BMRB data in NMR-STAR format.

Defaults

bmrb.display(version=`3.1')

Keyword arguments

version: The version of the BMRB NMR-STAR format to display.

Description

This will print the BMRB NMR-STAR formatted data to STDOUT.
