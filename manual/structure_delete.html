structure.delete

Synopsis

Delete structural information.

Defaults

structure.delete(atom_id=None)

Keyword arguments

atom_id: The atom identification string.

Description

This will delete structural information from the current data pipe. All spin and sequence information loaded from these structures will be preserved - this only affects the structural data. The atom ID argument can be used to restrict deletion to parts of the loaded molecules.

Prompt examples

To delete everything, simply type: 

[numbers=none]
relax> structure.delete()

To delete residues 50 to 100 of the molecule called `Ap4Aase', type one of: 

[numbers=none]
relax> structure.delete(':50-100')


[numbers=none]
relax> structure.delete(atom_id=':50-100')
