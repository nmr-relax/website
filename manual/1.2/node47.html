Initialisation of the run

Firstly to simplify referencing of the run name in the relevent functions the name `noe' is assigned to to the object name by the command


name = `noe'

Therefore instead of typing `noe' each time the run needs to be referenced, name can be used instead. The run is created by the command


run.create(name, `noe')

This user function will then create a run which is named `noe', the second argument setting the run type to that of calculating the NOE. Setting the run type is important so that the program knows which user functions are compatible with the run, for example the function minimise() is meaningless in this sample script as the NOE values are directly calculated rather than optimised.
