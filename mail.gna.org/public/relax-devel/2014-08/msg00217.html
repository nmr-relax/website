Details:

When creating a PDB representation of the diffusion tensor with
structure.create_diff_tensor, the resulting PDB contains a geometric object
wich represents the rotational diffusion tensor inferred from the relaxation
data. 

In case of the spheroidal or ellipsoid diffusion tensors, the diffusion axes
are drawn seperately and aligned to the structure as well as to the tensor
object. 

Except when it doesn't: for all except for the prolate diffusion tensor the
PDB is correctly rendered. The prolate PDB has the axis aligned to the long
axis of the molecule, but pointing somewhere into space and not aligned to the
long axis of the prolate spheroid object.

Please see if you can reproduce this, if not I can send you an results file in
private. 

Please see the attached rendering for clarification.
