rdc.write

Synopsis

Write the RDC data to file.

Defaults

rdc.write(self, id=None, file=None, dir=None, force=False)

Keyword Arguments

id: The alignment identification string.

file: The name of the file.

dir: The directory name.

force: A flag which if True will cause the file to be overwritten.

Description

If no directory name is given, the file will be placed in the current working directory. The `id' argument are required for selecting which RDC data set will be written to file.
