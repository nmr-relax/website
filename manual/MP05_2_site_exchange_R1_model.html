The MP05 2-site exchange R1 model

This is the model for 2-site exchange for off-resonance R1-type data for all time scales from Miloushev and Palmer (2005). It is selected by setting the model to `MP05'. The equation for the exchange process is

R1 = R1cos2 + R1'sin2    
  (theparentequation.66)

in which the parameters are defined as in the `TP02' model above. This model supersedes both the `TP02' and `TAP03' models. The reference for this equation is:

  • MiloushevPalmer05
