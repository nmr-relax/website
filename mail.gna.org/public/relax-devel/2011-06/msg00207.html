Hi Han,

You should now have commit access to the relax repository.  If anyone
has an objection, or if problems arise, this access can be revoked
later.  You will need to check out a new copy of relax using:

svn co svn+ssh://bugman@xxxxxxxxxxx/svn/relax/branches/xyz

replacing bugman with your Gna! login name.  Oh, before this you will
need to register an ssh key for secure access.  See
https://gna.org/cookbook/?func=detailitem&item_id=102 for how to do
this.
