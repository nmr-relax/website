J(w) mapping script mode - relaxation interactions

Prior to calculating the J() values, the physical interactions which govern relaxation of the spins must be defined. For the magnetic dipole-dipole relaxation interaction, the user functions are:

dipole_pair.define(spin_id1=`@N', spin_id2=`@H', direct_bond=True)
dipole_pair.set_dist(spin_id1=`@N', spin_id2=`@H', ave_dist=1.02 * 1e-10)

For the chemical shift relaxation interaction, the user function call is:

value.set(val=-172 * 1e-6, param=`csa')
