deselect.all

Synopsis

Deselect all spins in the current data pipe.

Defaults

deselect.all()

Description

This will deselect all spins, irregardless of their current state.

Prompt examples

To deselect all spins, simply type: 

[numbers=none]
relax> deselect.all()
