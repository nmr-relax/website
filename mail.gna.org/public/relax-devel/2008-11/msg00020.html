Greetings,

In running the full_analysis.py script for the ‘sphere’ model, I have
encountered an error during optimization of the diffusion tensor, primarily
that the size of the grid search is too large (see attached traceback and
output logs).  I am using experimental input data of my own (~246 residues)
that has been formatted to match the test files that I have used previously.

specify ‘sphere’ as the diffusion model.  This error does not occur when I
run the test files from /test_suite/shared_data/model_free/S2…0.149, so I am
not sure what is going on.  The reference pdb I am using is the output of the
Palmer r2r1_diffusion script.
