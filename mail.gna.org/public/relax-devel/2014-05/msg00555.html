Details:

To replicate this, simply do:

- Open the GUI.
- Select the spectrum.read_intensities user function from the menus.
- Type in the file name 'test'.
- Click on the file selection button.
- Close the window.

The file selection window will consist of 4 entries with the file names 't',
'e', 's', and 't'.  Closing the window will replace the name with ['t', 'e',
's', 't'].
