have been added.

This is for the 'TP02' model and 'NS R1rho 2-site' models.  The cpmg_fit 
results include source code
modifications to show the differences between the various 'corrections'.  The 
dispersion software
comparison file has been updated to include this data and to show the 
cpmg_fit verses relax
differences.
