Hi,

This one is quite easy:

$ relax -s Peak_lists

That will run all of the tests in that TestCase class.  If you would
like a specific test, say 'test_read_peak_list_nmrview', you would
use:

$ relax -s Peak_lists.test_read_peak_list_nmrview
