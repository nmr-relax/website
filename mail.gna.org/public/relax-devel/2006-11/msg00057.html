Details:

This problem appears to have been introduced between the 1.2.4 and 1.2.5
relax versions.  If model selection between the local tm models, spherical
diffusion, prolate spheroidal diffusion, oblate spheroidal diffusion, and
ellipsoidal diffusion is attempted, the local tm model is selected as only
the first residue of the local tm model is used in the model selection
algorithm.  This causes the 'full_analysis.py' script to fail!!!
