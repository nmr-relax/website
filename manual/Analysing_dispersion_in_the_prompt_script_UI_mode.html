Analysing dispersion in the prompt/script UI mode

Before reading this section, please read Chapter 4 covering the relax data model first. It will explain many of the concepts used within the following example script. For detailed information on how to run a relax script, please see section 1.2.8 on page . The dispersion analysis is parallelised on the level of the spin cluster and Monte Carlo simulations so, if you have access to an MPI cluster or multi-core system with OpenMPI installed, please see section 1.3 on page  for how to run the calculations much quicker.
