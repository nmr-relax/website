Compilation of the user manual (HTML version)

The HTML version of the relax user manual is made by typing


$ scons user_manual_html

in the base directory. One command calling the program latex2html will be executed which will create HTML pages from the LATEX sources.


The relax user manual (PDF), created 2015-10-15.
