vmd.view

Synopsis

Function for viewing the collection of molecules extracted from the PDB file.

Defaults

vmd.view(self, run=None)

Keyword Arguments

run: The name of the run which the PDB belongs to.

Example


relax> vmd.view(`m1')


relax> vmd.view(run=`pdb')
