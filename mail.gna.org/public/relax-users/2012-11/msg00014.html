Hi Kozo,

The file called 'log' will be located in the directory from which you
ran that command in, as you didn't specify which directory it will
created in.  This is assuming you have write permission to that
directory, otherwise a RelaxError will be produced saying:

RelaxError: Cannot open the file '/wrong_place_to_put_a_file'.
Permission denied.
