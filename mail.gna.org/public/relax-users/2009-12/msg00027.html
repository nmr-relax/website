Hi,

The directory m4 corresponds to the results of your analysis using model 4 from the modelfree models.

As is, the grace.py script reads the results of previous analysis for the model 4 of model free models located at the directory /m4.

Try going into relax and type help (grace.write) to understand what each parameter means. Anyway it seems to me that you should change the ‘S2’ (capital letter) to ‘s2’.

Also, bear in mind that the script that you are using perfoms the analysis considering only model 4 of the modelfree models.
