j_coupling.delete

Synopsis

Delete the J coupling values.

Defaults

j_coupling.delete()

Description

This will delete all J coupling data in the current data pipe.

Prompt examples

To delete all J coupling data, type: 

[numbers=none]
relax> j_coupling.delete()
