Hi Michael,

This is quite useful!  For the features
(http://www.nmr-relax.com/features.html), maybe it would be worth
including full-resolution images as links for the user to look at?
The current layout of images could be improved a little, on my machine

could be placed one after the other, in a column of screenshots?
Maybe something like http://inkscape.org/screenshots/index.php?lang=en

formatted as <p><em>$ relax --gui</em></p> (see
http://www.nmr-relax.com/docs.html where the relax prompt is
replicated).
