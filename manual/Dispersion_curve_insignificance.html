Dispersion curve insignificance

To avoid severe model failure due to the fitting of statistically insignificant dispersion curves, the relax_disp.insignificance user function can be used. This is activated by default in the auto-analysis. The user function takes a single insignificance value and if the difference between the smallest and largest R2eff or R1 value for an individual spin for all dispersion curves is less than this, then that spin will be deselected.
