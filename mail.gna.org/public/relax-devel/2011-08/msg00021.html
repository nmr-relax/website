Hi Han,
Be very careful! You only need to initialise svnmerge once at the very start on a branch, not every time you want to merge in the changes from the main line. For updating a branch, you only need to 
type 2 commands:

$ svnmerge.py merge

$ svn ci -F svnmerge-commit-message.txt
