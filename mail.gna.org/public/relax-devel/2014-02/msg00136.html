Details:

Steps to reproduce:

1.  Start the GUI mode.
2.  Create an NOE analysis using the new analysis wizard.
3.  Create a R1 analysis using the new analysis wizard.
4.  Click on the NOE analysis tab.
5.  Close the NOE analysis (File->Close analysis).

After these steps, the GUI will be displaying the R1 analysis tab, but the
current data pipe (visible in the bottom right of the screen) is blank.  It
looks like the R1 analysis is active but, because the current data pipe is set
to None, the R1 tab is not usable.  A simple fix is to use the pipe editor
window to set the R1 data pipe to be the current pipe.

This problem does not occur if the R1 tab is closed instead.  The behaviour
causes the NOE data pipe to become active.
