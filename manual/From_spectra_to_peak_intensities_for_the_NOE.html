From spectra to peak intensities for the NOE

For a set of recommendations for how to obtain the best quality relaxation rates, please see section 5.2 on page . In summary the following are important - temperature control (though the standard steady-state NOE single FID interleaved pulse sequences are fine), per-experiment temperature calibration, spectral processing with massive zero-filling and no baseplane rolling, and using an averaged peak list for determining the peak heights.
