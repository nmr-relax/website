Access to the internals of relax

To enable advanced Python scripting and control almost every part of relax has been designed in an object oriented fashion. If you would like to play with internals of the program the entirety of relax is accessible within the object called `self.relax'. To access the raw objects within relax which contain the program data the object called `self.relax.data' stores all the data.
