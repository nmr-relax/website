Posted by Edward d'Auvergne on June 09, 2011 - 14:34:
Hi,

I would suggest that rather than breaking up the page, we should have
a menu at the top which can be used to jump to each section.  I would
maybe include a new section about the SVN download after the source
release, and in that section warn that the code could be at points
unstable.  There would need to be the 'svn co' commands replicated,
and a reference to the 'svn up' command.  See for example the text at
https://mail.gna.org/public/relax-users/2006-11/msg00006.html which
can be copied and pasted for such a description.  It might need a link
to the subversion website.

Then for the Mac OS X section, I would suggest 3 ways of getting the
code.  The first would be to get the source code release (and link to
that section).  The second would be svn, with a link again.  And
finally Fink, as that is very out of date.  Jack, if you receive this,
what do you think?

Cheers,

Edward
