j_coupling.display

Synopsis

Display the J coupling data in the current data pipe.

Defaults

j_coupling.display()

Description

This will display all of the J coupling data in the current data pipe.

Prompt examples

To display all J coupling data, type: 

[numbers=none]
relax> j_coupling.display()
