Details:

This affects analyses whereby no 3D structure is available.  This bug is now
caught by the Mf.test_bug_20479_gui_final_pipe GUI test:

$ relax --gui-test Mf.test_bug_20479_gui_final_pipe

The problem is that the protocol fails if the prolate and oblate spheroids or
ellipsoid results cannot be found.  It is not known when this bug was
introduced as unfortunately this data combination was previously never tested
in the relax test suite.
