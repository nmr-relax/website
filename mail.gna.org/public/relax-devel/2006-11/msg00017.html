On 11/1/06, Douglas Kojetin <douglas.kojetin@xxxxxxxxx> wrote: 
I've been using the 1.2 stable branch.  What svn command should I use
to retrieve the updated code that interacts with PyMOL?


You need to have subversion installed.  Then you type:

svn co svn://svn.gna.org/svn/relax/branches/tensor_pdb

If that doesn't work, replace the 'svn://' with 'http://'.
