source code 

Spectral density function.

Calculate the spectral density values for the original model-free formula with no parameters {} with or without diffusion tensor parameters.

The formula is:

               _k_
            2  \                1
   J(w)  =  -   >  ci . ti ------------.
            5  /__         1 + (w.ti)^2
               i=-k
