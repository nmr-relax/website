Log:
Created a frame order rotor test model whereby the rotation axis is 
perpendicular to the domain.

The rotation is a total of 60 degrees about an axis perpendicular to the 
pivot-CoM vector.  The
'rotor' test model directory is reserved for a model whereby the rotation 
axis is the pivot-CoM
vector.
