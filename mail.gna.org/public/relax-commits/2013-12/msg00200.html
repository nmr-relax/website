Log:
Created synthetic R1rho dispersion data for the 'NS R1rho 3-site' model.

This is a simple modification of the data for the 'NS R1rho 3-site linear' 
model.  The k_AC
parameter was simply changed from 0 to 1000.  The cpmg_fit software was used 
to create the data.
Both cpmg_fit and relax results have been updated to the new model.
