The TP02 2-site exchange R1ρ model

This is the model for 2-site exchange for off-resonance R1ρ-type data from Trott and Palmer (2002). It is selected by setting the model to `TP02'. The equation for the exchange process is

R1ρ = R1cos2θ + R1ρ'sin2θ + , (11.76)

The equation is accurate only when populations are highly skewed with pA pB. And it is valid only for exchange processes which are not fast. Note that this model has been superseded by the `TAP03' and `MP05' models. The reference for this equation is:

  • Trott and Palmer (2002)

More information about the TP02 model is available from:

  • the relax wiki at http://wiki.nmr-relax.com/TP02,
  • the API documentation at http://www.nmr-relax.com/api/3.2/lib.dispersion.tp02-module.html,
  • the relaxation dispersion page of the relax website at http://www.nmr-relax.com/analyses/relaxation_dispersion.html#TP02.
