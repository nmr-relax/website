+     <h2>relax-announce</h2>
+      <p>The relax announcement list, relax-announce at gna.org, is reserved

list by vising the <a 
href="https://mail.gna.org/listinfo/relax-announce/";>relax-announce 
information page</a>. Previous announcements can be viewed in the <a 
href="https://mail.gna.org/public/relax-announce/";>archives</a>.</p>
+    </div>
+
+    <div id="relax-users" class="main_h2">
+     <h2>relax-users</h2>
+      <p>If you would like to ask questions about relax, discuss certain 
features, receive help, or to communicate on any other subject related to 
relax, the mailing list <a href="mailto:relax-users at gna.org">relax-users 
at gna.org</a> is the place to post your message.  To subscribe to the list, 
go to the <a href="https://mail.gna.org/listinfo/relax-users/";>relax-users 
information page</a>.  You can also <a 
href="https://mail.gna.org/public/relax-users/";>browse</a> the mailing list 
archives.</p>
+    </div>
+
+    <div id="relax-devel" class="main_h2">
+     <h2>relax-devel</h2>
+      <p>Another mailing list exists for posts relating to the development 
of relax.  The list is <a href="mailto:relax-devel at gna.org">relax-devel at 
gna.org</a> and to subscribe, go to the <a 
href="https://mail.gna.org/listinfo/relax-devel/";>relax-devel information 
page</a>.  Feature requests, program design, or any other posts relating to 
relax's structure or code should be sent to this list instead.  You can also 
<a href="https://mail.gna.org/public/relax-devel/";>browse</a> the mailing 
list archives.</p>
+    </div>
+
+    <div id="relax-commits" class="main_h2">
+     <h2>relax-commits</h2>
+      <p>One last mailing list is the relax commits list.  This list is 
reserved for automatically generated posts created by the version control 
software which looks after the relax source code and these web pages.  If you 
would like to become a developer, you can subscribe to the list at <a 
href="https://mail.gna.org/listinfo/relax-commits/";>relax-commits information 
page</a>.  The list can also be <a 
href="https://mail.gna.org/public/relax-commits/";>browsed</a>.</p>
+    </div>
+
+    <div class="main_h2">
+     <h2>Responding to messages</h2>
+      <p>Remember to hit 'respond to all' so that the mailing list is 
included in the CC field when replying to a message on these lists.  
Otherwise your message will only be sent to the original poster and will not 
return to the list.  Posts will only be accepted if sent to the relax-users 
or relax-devel mailing lists.</p>
+    </div>
