Posted by Paul Schanda on July 17, 2013 - 15:36:
Hi Edward,

No, there is nothing published. Nikolai said that anyways it's trivial :-)
He derived it basically by putting the Bloch-McConnell equations to Maple and simplifying it there. I guess there is something like a FullSimplify (that's Mathematica-style, but I guess Maple does it similarly).
I guess the main advantage is speed, and in fact in all cases I have seen it does exactly the same as the explicit Bloch-McConnell treatment, so I see it as the treatment that one would like to use. I agree that having something undocumented there is not particularly nice, but in the end it's up to the user whether or not to use this faster treatment (and compare it to the slower Bloch-McConnell one if needed). 

paul
