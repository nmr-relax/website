The NS MMQ 3-site linear model

This is the numerical model for 3-site exchange for proton-heteronuclear SQ, ZQ, DQ and MQ CPMG data, as derived in (Korzhnev et al., 2005a,2004a,b). As this model is linear, the assumption that kAC = kCA = 0 has been made. To simplify the optimisation space for the model, the assumption R2A0 = R2B0 = R2C0 = R20 has also been made.
