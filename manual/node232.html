nuclei

Synopsis

Function for setting the gyromagnetic ratio of the heteronucleus.

Defaults

nuclei(self, heteronuc=`N')

Keyword arguments

heteronuc: The type of heteronucleus.

Description

The heteronuc argument can be set to the following strings:

N -
Nitrogen, -2.7126e7
C -
Carbon, 2.2e7
