Follow-up Comment #61:

This branch is discontinued.

This is due to:
http://article.gmane.org/gmane.science.nmr.relax.scm/24758

Since relax is not able in its current form to efficiently store numpy arrays,
within a fast timeframe, it gives no meaning to continue this development.

Therefore this is discontinued.

On top of this, relax cannot save Class instance objects.
This gives a problem, when not being able to store the returned axis instance
from matplotlib.

Which means that any additional decoration to the axis plot, is lost upon
saving.
