molmol.ribbon

Synopsis

Apply the Molmol ribbon style.

Defaults

molmol.ribbon(self, run=None)

Keyword Arguments

run: The name of the run.

Description

This function applies the Molmol ribbon style which is equivalent to clicking on `ribbon' in the Molmol side menu. To do this, the following commands are executed:

CalcAtom `H'
CalcAtom `HN'
CalcSecondary
XMacStand ribbon.mac

Example

To apply the ribbon style to the PDB file loaded for the run `final', type:


relax> molmol.ribbon("final")
