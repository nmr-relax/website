 82      """Calculate the average distances. 
 83   
 84      The formula used is:: 
 85   
 86                    _N_ 
 87                / 1 \                  \ 1/exp 
 88          <r> = | -  > |p1i - p2i|^exp | 
 89                \ N /__                / 
 90                     i 
 91   
 92      where i are the members of the ensemble, N is the total number of structural models, and p1 
 93      and p2 at the two atom positions. 
 94   
 95   
 96      @param atom1:   The atom identification string of the first atom. 
 97      @type atom1:    str 
 98      @param atom2:   The atom identification string of the second atom. 
 99      @type atom2:    str 
100      @keyword exp:   The exponent used for the averaging, e.g. 1 for linear averaging and -6 for 
101                      r^-6 NOE averaging. 
102      @type exp:      int 
103      @return:        The average distance between the two atoms. 
104      @rtype:         float 
105      """ 
106   
107      # Get the spin containers. 
108      spin1 = return_spin(atom1) 
109      spin2 = return_spin(atom2) 
110   
111      # Loop over each model. 
112      num_models = len(spin1.pos) 
113      ave_dist = 0.0 
114      for i in range(num_models): 
115          # Distance to the minus sixth power. 
116          dist = norm(spin1.pos[i] - spin2.pos[i]) 
117          ave_dist = ave_dist + dist**(exp) 
118   
119      # Average. 
120      ave_dist = ave_dist / num_models 
121   
122      # The exponent. 
123      ave_dist = ave_dist**(1.0/exp) 
124   
125      # Return the average distance. 
126      return ave_dist
