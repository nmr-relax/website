 36      Equations

 51   
 52          3a'*alpha**2 + 2b'*alpha + c' = 0 
 53   
 54      The cubic interpolant is given by the formula:: 
 55

 64                                       a - b 
 65

 68              beta2 = sqrt(beta1**2 - g(a).g(b)) 
 69

 71
