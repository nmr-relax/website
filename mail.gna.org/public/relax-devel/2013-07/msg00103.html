Hi Paul, Mathilde and Dominique,

I would like to finalise the copyright notices in your code that is now in relax.  This is in the lib/dispersion/ns_*.py files.  If you have the subversion program installed on your machine (all operating systems are supported), you can get the code by typing at the terminal, cmd command prompt, etc:

$ svn co http://svn.gna.org/svn/relax/branches/relax_disp

At the end of this message, you will see the copyright statements from these files.  But note that I have guessed the dates.  And I have guessed who added or modified this code.  Please carefully check the dates and if your name should be in the file - this information is incredibly important.

Cheers,

Edward
