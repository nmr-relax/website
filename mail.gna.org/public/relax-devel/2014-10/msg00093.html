Posted by Edward d'Auvergne on October 14, 2014 - 12:50:
Hi Troels,

If you would like to see if optimising the R20A parameters and dw for
a single spin will fix this (i.e. just the spin-dependent parameters),
a simple way would be to make some temporary changes that you never
commit to the relax repository.  You could hard code the k_AB
parameter directly into the target function to the optimised value you
now have.  And then remove it from the parameter list of the model.
Then turn off the clustering and optimise just this spin (deselecting
all others).  You will quickly see if the Nelder-Mead simplex
algorithm can slide along this curved and narrow tunnel in the R20A+dw
space.

Regards,

Edward
