Log:
Added a script and log for calculating the exponential curve parameter errors 
via bootstrapping.

This uses the data at 
http://thread.gmane.org/gmane.science.nmr.relax.devel/6807/focus=6840 and
calculates the parameter errors via bootstrapping.  As the parameters at the 
minimum are the exact
parameter values, bootstrapping and Monte Carlo simulation converge and hence 
this is a true error
estimate.  200,000 simulations where used, so the parameter errors are 
extremely accurate.
