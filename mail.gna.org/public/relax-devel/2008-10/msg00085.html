Hi,

It seems there is no default log file name. Hence, if one doesn't specify a
log file name when using the '-l' or '-t' option, the script used will be
erased...

Example
=======

relax -l script.py  -> This sends the log in the file 'script.py'.

relax -t script.py  -> This sends the log in the file 'script.py'.
