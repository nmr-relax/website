About relax-devel

This is the relax-devel mailing list for the relax NMR software project. It is a list reserved for the development of relax. To see the collection of prior postings to the list, visit the relax-devel Archives.

Gna! shutdown
The Gna! open source infrastructure was shut down at the start of 2017. Hence the relax-devel@gna.org mailing list no longer exists. This page is a copy of the original, modified for archival purposes. It was taken from the Internet Archive.
