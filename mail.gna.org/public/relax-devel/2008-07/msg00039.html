Hi,

Thanks for making all these fixes for the Boolean flags and the
cleaning up of the sample scripts.  There's quite a lot of old style
code that needs this kind of clean up in the 1.3 line.  In the case of
the heteronucleus, one other important change that has happened in the
redesign is that this variable is spin system specific.  This is to
allow different types of spins to be analysed simultaneously.
Therefore this user function should be executed only after the
molecule-residue-spin sequence has been loaded.  I might make these
changes another day (if they haven't already been fixed ;).

Cheers,

Edward
