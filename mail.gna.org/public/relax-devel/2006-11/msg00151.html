Hi,

I've added a new webpage (http://nmr-relax.com/links.html) with links
to websites where relax can be found on the web.  It might take a
couple of hours to appear.  On another note, the relax website sources
are have been shifted into the directory 'website' of the SVN
repository.  The current links are:
* Gna! (http://gna.org/) where relax is housed.
* The Mail Archive (http://mail-archive.com/) which archives the relax
mailing lists.  We should configure this site a little, especially to
get the old mbox files located in the mailing list archives under the
link called 'Full archive gzipped'.
* freshmeat (http://freshmeat.net/) where relax is announced (together
with the news page https://gna.org/news/?group=relax and the
relax-announce mailing list).
* CIA (http://cia.navi.cx/) where relax commit messages are stored and
relax coding stats are collated.
* LinuxLinks.com where there is a link to relax.

If anyone would like to help increase relax's search engine rankings
and indexing (especially of the mailing list archives to help with
their searching) by increasing the number of links to
http://nmr-relax.com, it would be appreciated.  Hopefully, one day,
Gna! will add proper mailing list searching capabilities to their
infrastructure.

Cheers,

Edward
