Hi Paul,

I think you'll find the solution for the numpy problem at
http://wiki.nmr-relax.com/Multiple_Python_versions.  My guess for the
scons problem could be because the code was checked out as root.  I
would suggest having the relax_disp branch checked out as a user.  You
can always create a symbolic link in ~/bin to this.  Of if this is for
multiple users, then in /usr/local/bin.

Regards,

Edward
