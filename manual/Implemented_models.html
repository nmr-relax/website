Implemented models

A number of analytic and numeric models are supported within relax. These cover single quantum (SQ) CPMG-type, single quantum (SQ) R1-type, and combined proton-heteronuclear single quantum (SQ), zero quantum (ZQ), double quantum (DQ) and multi quantum (MQ) CPMG-type experiments. If the model you are interested in is not available, please see Section 10.11 on page  for how you can add new models to relax.

Models which are independent of the experiment type include:

`R2eff':
This is the model used to determine the R2eff or R1 values and errors required as the base data for all other models. See Section 10.2.1 on page .
`No Rex':
This is the model for no chemical exchange being present. See Section 10.2.2 on page .

For the SQ CPMG-type experiments, the analytic models currently supported are:

`LM63':
The original Luz and Meiboom (1963) 2-site fast exchange equation with parameters {R20,...,ex, kex}. See Section 10.3.1 on page .
`LM63 3-site':
The original Luz and Meiboom (1963) 3-site fast exchange equation with parameters {R20,...,ex, B, kB,ex, C, kC}. The equations of O'Connell et al. (2009) can be used to approximately convert the parameters {ex, B, kB,ex, C, kC} to more biologically relevant parameters. See Section 10.3.2 on page .
`CR72':
The reduced Carver and Richards (1972) 2-site equation for most time scales whereby the simplification R2A0 = R2B0 is assumed. It has the parameters {R20,..., pA,, kex}. See Section 10.3.4 on page .
`CR72 full':
The full Carver and Richards (1972) 2-site equation for most time scales with parameters {R2A0, R2B0,..., pA,, kex}. See Section 10.3.3 on page .
`IT99':
The Ishima and Torchia (1999) 2-site model for all time scales with pA pB and with parameters {R20,...,ex, pA., kex}. See Section 10.3.5 on page .
`TSMFK01':
The Tollinger et al. (2001) 2-site very-slow exchange model for time scales within range of microsecond to second time scale. Applicable in the limit of slow exchange, when | R2A0 - R2B0| kAB, kBA 1/CPMG. 2*CPMG is the time between successive 180 degree pulses. Parameters are {R2A0,...,, kAB}. See Section 10.3.6 on page .

For the SQ CPMG-type experiments, the numeric models currently supported are:

`NS CPMG 2-site expanded':
A model for 2-site exchange expanded using Maple by Nikolai Skrynnikov (Tollinger et al., 2001). It has the parameters {R20,..., pA,, kex}. See Section 10.4.1 on page .
`NS CPMG 2-site 3D':
The reduced model for 2-site exchange using 3D magnetisation vectors whereby the simplification R2A0 = R2B0 is assumed. It has the parameters {R20,..., pA,, kex}. See Section 10.4.3 on page .
`NS CPMG 2-site 3D full':
The full model for 2-site exchange using 3D magnetisation vectors with parameters {R2A0, R2B0,..., pA,, kex}. See Section 10.4.2 on page .
`NS CPMG 2-site star':
The reduced model for 2-site exchange using complex conjugate matrices whereby the simplification R2A0 = R2B0 is assumed. It has the parameters {R20,..., pA,, kex}. See Section 10.4.5 on page .
`NS CPMG 2-site star full':
The full model for 2-site exchange using complex conjugate matrices with parameters {R2A0, R2B0,..., pA,, kex}. See Section 10.4.4 on page .

For the combined proton-heteronuclear SQ, ZQ, DQ and MQ CPMG-type experiments (MMQ - or multi-multiple quantum), the analytic models currently supported are:

`MMQ CR72':
The Carver and Richards (1972) 2-site model for most time scales expanded for MMQ CPMG data by Korzhnev et al. (2004a). It has the parameters {R20,..., pA,,, kex}. See Section 10.5.1 on page .

For the combined proton-heteronuclear SQ, ZQ, DQ and MQ CPMG-type experiments (MMQ - or multi-multiple quantum), the numeric models currently supported are:

`NS MMQ 2-site':
The model for 2-site exchange whereby the simplification R2A0 = R2B0 is assumed. It has the parameters {R20,..., pA,,, kex}. See Section 10.6.1 on page .
`NS MMQ 3-site linear':
The model for 3-site exchange linearised with kAC = kCA = 0 whereby the simplification R2A0 = R2B0 = R2C0 is assumed. It has the parameters { R20, ..., pA, pB, AB, BC, AB, BC, kexAB, kexBC}. See Section 10.6.2 on page .
`NS MMQ 3-site':
The model for 3-site exchange whereby the simplification R2A0 = R2B0 = R2C0 is assumed. It has the parameters { R20, ..., pA, pB, AB, BC, AB, BC, kexAB, kexBC, kexAC}. See Section 10.6.3 on page .

For the R1-type experiments, the analytic models currently supported are:

`M61':
The Meiboom (1961) 2-site fast exchange equation for on-resonance data with parameters {R1',...,ex, kex}. See Section 10.7.1 on page .
`DPL94':
The Davis et al. (1994) extension of the `M61' model for off-resonance data with parameters {R1',...,ex, kex}. See Section 10.7.3 on page .
`M61 skew':
The Meiboom (1961) 2-site equation for all time scales with pA pB and with parameters {R1',..., pA,, kex}. This model is disabled by default in the dispersion auto-analysis. See Section 10.7.2 on page .
`TP02':
The Trott and Palmer (2002) 2-site equation for all time scales with pA pB and with parameters {R1',..., pA,, kex}. See Section 10.7.4 on page .
`TAP03':
The Trott et al. (2003) off-resonance 2-site analytic equation for all time scales with the weak condition pA pB and with parameters {R1',..., pA,, kex}.
`MP05':
The Miloushev and Palmer (2005) off-resonance 2-site equation for all time scales with parameters {R1',..., pA,, kex}. See Section 10.7.6 on page .

For the R1-type experiments, the numeric models currently supported are:

`NS R1rho 2-site':
The model for 2-site exchange using 3D magnetisation vectors. It has the parameters {R1',..., pA,, kex}. See Section 10.8.1 on page .
`NS R1 3-site linear':
The model for 3-site exchange linearised with kAC = kCA = 0 whereby the simplification R1A' = R1B' = R1C' is assumed. It has the parameters { R1', ..., pA, pB, AB, BC, kexAB, kexBC}. See Section 10.8.3 on page .
`NS R1 3-site':
The model for 3-site exchange whereby the simplification R1A' = R1B' = R1C' is assumed. It has the parameters { R1', ..., pA, pB, AB, BC, kexAB, kexBC, kexAC}. See Section 10.8.2 on page .
