system.pwd

Synopsis

Display the current working directory.

Defaults

system.pwd()

Description

This will display the current working directory.

The directory can be changed with the system.cd(path) user function. 

[numbers=none]
relax> system.pwd()


[numbers=none]
relax> system.cd("/path/to/dir")
