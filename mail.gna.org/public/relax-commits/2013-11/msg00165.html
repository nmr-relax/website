Log:
Different relaxation time periods for each experiment is now taken into 
account in the dispersion code.

Previously only the first relaxation time period was being used.  This was 
fine for single data type
models, but was preventing the MMQ-type models from working.  Now the 
return_r2eff_data() function
of the specific_analyses.relax_disp.disp_data module assembles and returns 
the relax_times data
structure which has two dimensions - the experiment type and the magnetic 
field strength.
