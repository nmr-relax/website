Log:
Change of the base relaxation dispersion experiment types.

The base CPMG-type experiment has been changed from "CPMG" to "SQ CPMG".  
This is for better
combined proton-heteronuclear SQ, ZQ, DQ, and MQ (MMQ) data support.  The 
relax_disp.exp_type user
function now also as the proton SQ and MQ CPMG-types available to select from 
rather than the
previous behaviour of relax automatically determining the type from the spin 
type.

All of the CPMG experiment type variables in 
specific_analyses.relax_disp.variables have been
renamed for better ordering.  Many changes were therefore required.
