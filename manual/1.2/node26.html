Dependencies

The following packages need to be installed before using relax:

http://python.org/Python:
Version 2.4 or higher (although any 2.x version may work).
http://numpy.scipy.org/Numeric:
Version 21 or higher.
http://starship.python.net/ hinsen/ScientificPython/ScientificPython:
Version 2.2 or higher.
Optik:
Version 1.4 or higher. This is only needed if running python 2.2.

Older versions of these packages may work, use them at your own risk. If, for older dependency versions, errors do occur please submit a bug report to the bug tracker at https://gna.org/bugs/?group=relaxhttps://gna.org/bugs/?group=relax. That way a solution may be created for the next relax release.
